Employee Profile Module

A complete self-service employee portal with secure document management, automated HR letters, and real-time profile tracking.

New Feature HR Management Document Vault Self-Service Auto Letters
This update introduces a fully integrated Employee Profile system into MS HRMS. Every employee now has a dedicated profile page, a personal document vault, and the ability to request profile updates — all managed through a structured HR approval workflow.

✦ What's New

Employee Profile Dashboard

Each employee has a dedicated profile page showing their personal details, employment information, reporting manager, and current documents — all in one place.

Profile Completion Tracker

A visual progress ring shows how complete an employee's profile is. HR can instantly see who has missing documents or incomplete information across the organisation.

Secure Document Vault

Employees can upload KYC documents, identity proofs, banking details, and qualifications directly from their profile. All files are stored securely in the cloud — never accessible via a public link.

HR Verification Workflow

HR reviews each uploaded document and marks it as Verified or Rejected with a reason. Employees are notified immediately and can re-upload if required.

Self-Service Profile Updates

Employees can request changes to their phone number, address, bank details, or emergency contact. HR reviews and approves the change before it is applied — no direct editing.

Automated HR Letters

HR can generate official letters in one click — offer letters, appointment letters, salary increment letters, experience certificates, and more — directly from within the employee profile.

👥 Who Uses It & What They Can Do

Role Capabilities
Employee View their own profile, upload KYC & personal documents, download their letters, request profile corrections, track document verification status.
HR View all employee profiles, verify or reject documents with remarks, generate and issue official letters, approve or reject profile change requests, monitor profile completion across the team.
Manager View team members' profiles and employment documents. Cannot access salary or KYC documents of other employees.
Accounts Access salary-related documents only (payslips, salary structures, bank proofs) — no access to other document categories.
Admin Full access to all profiles, documents, letters, and change requests. Can manage letter templates for the organisation.

◎ Profile Completion Tracker

72%

Live completion percentage for every employee

The ring updates in real time as employees fill in details and upload documents. When HR adds a new required field in the future, all profiles automatically recalculate and employees are notified to complete the new information.

✉ Automated HR Letters

HR can generate any of the following letters in one click. The system auto-fills the employee's name, designation, salary, joining date, and other details from their profile. A unique reference number is assigned to every letter automatically.

Offer Letter Appointment Letter Confirmation Letter Promotion Letter Salary Increment Letter Warning Letter Experience Certificate Relieving Letter

Generated letters are saved automatically in the employee's document vault and are available for the employee to download from their profile.

🗂 Document Categories Supported

  • ✓ KYC Documents — Aadhaar card, PAN card, passport, driving licence, voter ID
  • ✓ Banking Documents — Bank account proof, cancelled cheque, passbook
  • ✓ Education & Employment — Degree certificates, previous offer letters, experience letters, relieving letters
  • ✓ Salary Documents — Payslips, salary structures, increment letters (restricted to HR and Accounts)
  • ✓ HR-Generated Letters — Stored automatically when HR issues any official letter

🔒 Security & Privacy

☁

Cloud Storage

All documents are stored securely in Supabase cloud storage. No file is ever served from a public URL.

🔗

Expiring Access Links

Download links are temporary and expire after 5 minutes. Each access is logged with a timestamp.

🛡

Role-Based Access

Accounts can only see salary documents. Managers cannot see KYC. Employees see only their own files.

📋

Full Audit Trail

Every document view, upload, verification, and letter generation is recorded with the user, date, and action.

✏

Approval Workflow

Employees cannot directly edit their own records. All changes go through HR review before being applied.

🔔

Expiry Notifications

The system automatically reminds HR and the employee 30 days before any document (e.g. passport) expires.
